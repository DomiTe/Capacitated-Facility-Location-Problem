ADM Semesterprojekt documentation
=================================

Diese Seite enthält die Dokumentation für das Semesterprojekt im Fach Anwendungen von diskreter Mathematik.
Das Projekt wurde von den Studierenden des Kurses im Sommersemester 2025 durchgeführt und ist in Python geschrieben.
Das Projekt ist in mehrere Module unterteilt, die verschiedene Algorithmen und Datenstrukturen implementieren.
